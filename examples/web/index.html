block-model-renderer

Minecraft block and item model rendering for Node.js and the browser.

Render any block, item, or custom model JSON, with full support for vanilla resource pack features.

GitHub NPM JSDelivr

Live Examples

Each example runs entirely in your browser. They download the latest vanilla Minecraft client jar directly from Mojang's servers, and you can load your own resource packs and mods on top of it.

Model Viewer

Load any block or item in an interactive three.js viewer. Blockstate properties, display transforms, lighting modes, face culling, animated textures, and wireframe view.

Render Gallery

Batch-render blocks straight to canvases with renderBlock, complete with live animation players. Filter by name or namespace, or render every block in the pack.

Rotating Grid

An endless scrolling wall of spinning blocks and items. Drag to pan, pinch or scroll to zoom, and tweak the motion from the controls panel.

Scene Builder

Little voxel dioramas built with the low-level API. Blocks are culled against their neighbours, fluids shape their surfaces, and the whole scene is merged into a handful of draw calls.

Features

  • Renders blocks, items, and custom models from a resource pack
  • Runs on Node.js and in the browser from the same package
  • Full vanilla model, blockstate, item-definition, and texture atlas support, with accurate lighting and tints
  • Bundled overrides for block entities that Minecraft renders dynamically (banners, chests, heads, etc)
  • Stack resource pack folders, zips, and virtual handlers, with higher packs overriding lower ones just like in Minecraft
  • Animated textures: WebP and GIF output on Node, live self-updating canvases on web
  • Scene optimisation: near game-accurate hidden-face culling from neighbouring blocks, and placed blocks merged into a few atlased, greedily meshed draw calls
  • Extensible model loaders: write your own to support modded formats (OBJ models, connected textures, etc)
  • PNG, JPEG, WebP, GIF, and AVIF output on Node

Getting Started

Install from NPM for Node.js, or the browser through a bundler:

npm install block-model-renderer

Or import it straight from a CDN in the browser:

import { renderBlock, prepareAssets } from "https://cdn.jsdelivr.net/npm/block-model-renderer@latest/+esm"

// a resource pack zip from a file input, fetch, anywhere
const assets = await prepareAssets([resourcePackZip])

// returns a canvas you can append
const canvas = await renderBlock({ id: "grass_block", assets, width: 128, height: 128 })
document.body.append(canvas)

The full documentation lives in the GitHub repository, covering rendering options, asset sources and pack layering, fluids, scene building, and custom model loaders.
